Homework 1: Assignments List

  • Homework 1: CNIT 133 Homework Page
  • Create a web page, using HTML5
  • Homework 2: Basic Operations
  • Create pages for scripts using document.write statements, and counts
  • Homework 3: Functions
  • Create scripts for calculating student's final average, total earnings by commisions, converting Celsius to Fahrenheit (and vice versa), and randomized multiplication questions
  • Homework 4: Looping Statements
  • Create scripts for calculations and compound interest
  • Homework 5: Arrays
  • Create scripts for different input and select boxes, validation, pulldown menus, and finding data
  • Homework 6: Objects
  • Create a webpage with the following scripts:
    (1) a script calculating square root and rounding floating-point numbers
    (2) a script using a form that allows the user to input lines of text and a search character, then determine the number of occurances of the search character within the text
    (3) use the jQuery mask for inputting a telephone number as a string in the format (999) 999-9999

  • Create a webpage with jQuery and the following scripts:
    (1) a script allowing the users to select page layout and formatting through the className property
    (2) a script displaying a top navigation bar for previous homeworks, and displaying a drop-down menu for different parts
    (3) use the jQuery mask for inputting a telephone number as a string in the format (999) 999-9999
