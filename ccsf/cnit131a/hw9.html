Homework 9: SVG Files in CNIT 131A

Above is the SVG file embedded in this HTML document as an image. It contains 4 shapes: a yellow rectangle with a blue border, a blue circle with a red border, a thin and long horizontal rectangle with a color gradient from yellow to blue, and a thin and long vertical rectangle with a color gradient from blue to red. The yellow rectangle used the rect element and the basic attributes for its styling:

                <rect x="60" y="10" fill="yellow" width="150px" height="120px" stroke="blue" stroke-width="2px"/>

The blue circle used the circle element and the basic attributes for its styling:

                <circle fill="blue" cx="290" cy="70" r="60px" stroke="red" stroke-width="2px"/>

Both thin rectangles use the defs element for the linear gradient fill. It allows for the defining of constants or special elements, which in this case is for specific gradient colors that are used for the fill colors in the rectangles:

                <defs>
                          <linearGradient id="LGx" x1="0%" y1="0%" x2="100%" y2="0%">
                            <stop offset = "0%" stop-color = "yellow" />
                            <stop offset = "100%" stop-color = "blue" />
                          </linearGradient>
                </defs>
            
                <defs>
                      <linearGradient id="LGy" x1="0%" y1="0%" x2="0%" y2="100%">
                        <stop offset = "0%" stop-color = "blue" />
                        <stop offset = "100%" stop-color = "red" />
                      </linearGradient>
                </defs>
                
                <rect x="60" y="150" width="290px" height="30px" fill="url(#LGx)" />
                <rect x="10" y="10" width="30px" height="170px" fill="url(#LGy)" />

It is important to note that the code is slightly different between the horizontal and vertical rectangles. Initially, the horizontal rectangle did not need the x and y attributes for the linearGradient element. It was only required for the vertical rectangle to define the coordinates. It was added to both gradients to provide futher information on defining its beginning and end points. According to W3Schools SVG Linear Tutorial, horizontal gradients are when the y coordinatese are equal and the x coordinates differ, and vertical gradients are when the x corrdinates are equal and the y coordinates differ.
